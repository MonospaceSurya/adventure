Optimal IRB140 January 2023

862 AI Medical Imaging Deep Learning Computer Vision Healthcare Tech

Optimizing ABB IRB 140 Robotic Arm with Shortest Path Algorithms in a 3D Simulation

This project features a 3D simulation of the ABB IRB 140, a 6-DOF robotic arm, designed using a Blender-modeled mockup. The simulation leverages shortest path algorithms, including Dijkstra's, to optimize the arm's movements for tasks such as picking and placing objects. The focus is on minimizing traversal path length, energy consumption, heat generation, and mechanical wear. By integrating realistic physics and motion constraints, the project delivers a robust simulation for efficient and precise robotic arm operation.

Technical Details:

  • Modeled ABB IRB 140 in Blender for realistic visuals.
  • Used Dijkstra’s and others for optimal path planning.
  • Minimized path length, energy use, and heat generation.
  • Visualized movements with Three.js and joint constraints.
  • Implemented obstacle detection for safe operation.

Image Colorization January 2023

862 AI Medical Imaging Deep Learning Computer Vision Healthcare Tech

Let there be Color!

This project was a deep dive into the fascinating world of image colorization, where I tackled the challenge of automatically adding realistic color to grayscale photographs. The core idea was to reframe this inherently ambiguous problem as a classification task within a deep learning framework. I trained a convolutional neural network on a massive dataset of color images, carefully balancing the color classes to encourage vibrant and diverse outputs.

Technical Details:

  • Utilized a CNN architecture as the backbone for image processing and feature extraction.
  • Formulated colorization as a classification problem, assigning color probabilities to each pixel.
  • Implemented a class-rebalancing strategy during training to enhance the diversity and vibrancy of the generated colors.
  • Trained the model on a dataset comprising over one million color images.
  • Leveraged Python along with TensorFlow/PyTorch for model development and training.

Glaucoma Detection January 2023

862 AI Medical Imaging Deep Learning Computer Vision Healthcare Tech

Advancing Early Diagnosis of Glaucoma through Deep Learning and Optic Disc Segmentation techniques

Glaucoma, a leading cause of irreversible blindness, necessitates innovative diagnostic solutions. By applying deep learning to fundus/retinal images, this project aimed to develop an advanced detection method for improved early glaucoma identification and diagnosis.

Technical Details:

  • Utilized Drishti-GS dataset for comprehensive glaucoma image analysis
  • Fine-tuned Segment Anything Model (SAM) for precise optic disc segmentation.
  • Systematically tested neural network architectures, with AlexNet achieving highest precision for glaucoma detection.
  • Implemented Explainable AI techniques to enhance interpretability for medical professionals.
  • Hardware Acceleration: Utilized an RTX 3050 GPU for efficient model training and evaluation.
